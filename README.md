# MGKeyGenerator

MGKeyGenerator é uma biblioteca para gerar chaves aleatórios com e sem hash.

---

## ✨ Características

* Zero dependências externas
* Compatível com **PHP 8.3 ou superior**

---

## 📦 Instalação

### Via Composer (recomendado)

```bash
composer require mugomes/mgkeygenerator
```

### Manual

Copie os arquivos da pasta `MGKeyGenerator` para o seu projeto e utilize `include_once`;
